Difference between FlexBox and Grid?

Flexbox is one dimentional, but Grid is two Dimentional. Flex is easy to use for any beginners, but Grid is lest easy than Flexbox. Althouh, Grid is more important than Flex, because we can do responsive easily with grid more than flexbox layout.

What is the Difference between Boostrap and Tailwind CSS?

The major Difference is, Bootstrap has limited utilites for Design and Development. But Tailwind has a lot of utilities and Class Components which we can use for our Daily work. Another difference is, Bootstrap creates components for us and its free. Tailwind has also, but they are need to pay to use in our projects.

What is CSS box model?

CSS box model is the kind of thing which can give us the information about margin, padding, border and actual contents. When we debug, it shows how much padding, margin, border and content we are using in model. Then we can handle or modify our design.

What is a Semantic Tag/Element?

Semantic elemnts are basically article, aside, deatails, blockquotes, figure, footer, main, mark etc tag which we use in our projects. They represents difference layout in a projects. They are the latest HTML 5 edition.
